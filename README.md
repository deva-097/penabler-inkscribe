# Penabler 🖋️

**Your Ultimate Fountain Pen & Ink Collection Manager**

Penabler is a comprehensive web application for fountain pen enthusiasts to track their pen and ink collections, manage inking schedules, and monitor pen maintenance. Built for collectors who want to stay organized and get the most out of their writing instruments.

## Features

### 📊 **Dashboard Overview**
- Quick stats: Total pens, inks, active pens, and neglected pens
- Insights: Most active pen, most used ink, longest inked pen
- Recent activity tracking

### 🖊️ **Pen Collection Management**
- Add and track fountain pens with detailed specifications
- Record nib sizes, materials, eras, and purchase information  
- Monitor pen status (inked, cleaned, neglected)
- Track total usage days per pen

### 🎨 **Ink Collection**
- Catalog inks with color swatches and properties
- Track ink characteristics: waterproof, sheen, shimmer, shading
- Volume and price tracking with multi-currency support
- Visual color representation

### 📝 **Inking Activity**
- Log inking events (inked, re-inked, cleaned)
- Complete inking history with notes
- Quick event logging from any page
- Track which ink is currently in each pen

### 🔍 **Smart Views**
- **Active Pens**: See currently inked pens and how long they've been inked
- **Neglected Pens**: Find pens unused for 90+ days
- **Collection Views**: Browse and search your entire collection

### 💰 **Multi-Currency Support**
- Track purchases in different currencies (USD, EUR, GBP, JPY, CAD, AUD)
- Consistent pricing display throughout the app

## Tech Stack

- React + TypeScript
- Supabase (PostgreSQL + Auth)
- shadcn-ui + Tailwind CSS
- Vite
